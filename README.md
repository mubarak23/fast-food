## Key Features

- Orders
  - Instantly see all orders.
  - Filter orders
  - Make multiple orders at once
  - Edit previous order before admin sees it
- Cart
  - Save item for later purchase
  - Enable use even when logged out
- Menu
  - Add review for a menu
  - Filter menu
- Admin
 - Add Food Item
 - Analytics for order Data
